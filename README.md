# Drift & Discover

A modern, high-converting 3D animated travel website built with Next.js, Tailwind CSS, Framer Motion, and Three.js.

## Getting Started

Since Node.js, npm, and Git were not found on your system, please install them first:
1.  **Node.js & npm:** [Download here](https://nodejs.org/)
2.  **Git:** [Download here](https://git-scm.com/)

Once installed, follow these steps to run the development server:
1.  Open a terminal in this directory (`d:/Tour/Pranav`).
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.

## Upload to GitHub

To upload this project to your GitHub account:
1.  Create a new repository on [GitHub](https://github.com/new).
2.  Open a terminal in this directory and run:
    ```bash
    git init
    git add .
    git commit -m "Initial commit: Drift & Discover 3D Travel Website"
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    git push -u origin main
    ```
    *(Replace YOUR_USERNAME and YOUR_REPO_NAME with your actual details)*
